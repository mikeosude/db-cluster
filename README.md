# Database Cluster Configurations

Database cluster configuration files and scripts for:
- **Percona XtraDB Cluster**
- **PostgreSQL**

## Structure

```
db-cluster/
├── percona-xtradb/   # Percona XtraDB Cluster configs
├── postgresql/       # PostgreSQL configs
└── scripts/          # Setup and maintenance scripts
```

## Usage

Clone and use the configurations as needed for your database setups.

```bash
git clone https://github.com/mikeosude/db-cluster.git
```

## Note

Do not commit sensitive credentials. Use environment variables for passwords and sensitive data.
